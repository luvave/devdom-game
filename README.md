## Usage

### Installation

```bash
$ yarn install
```

### Development

Run and visit http://localhost:5173

```bash
$ yarn start
```

### Build

To build the App, run

```bash
$ yarn build
```

### Preview

To preview the App, run

```bash
$ yarn preview
```

### Test

```bash
$ yarn test
```
